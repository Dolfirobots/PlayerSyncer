# Security Policy

## Plugin Version Support
All all recent bug fixes, security patches, and new features are only maintain in the lasted plugin version.

## Reporting a Vulnerability
If you discover a security vulnerability, please report it responsibly:  

1. **Do not create a public issue** on GitHub, only if the vulnerability is not high.  
2. **Open a ticket on our [Discord server](https://discord.gg/dxZTGpPbkd/)**
3. We will acknowledge your report promptly and provide updates until the issue is resolved.  

## Security Best Practices
- Always use the latest plugin version.  
- Follow official Minecraft server security guidelines.  
- Regularly back up your server and plugin data.  

## Acknowledgments
We appreciate responsible disclosure and thank security researchers who help us keep our plugin safe.
